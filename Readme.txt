NewsAutoRu bot for telegram
============================

Телеграм-бот NewsAutoRu позвроляет искать автомобильные новости по ключевому слову и самой ранней дате статьи.
Для этого нужно отправить боту комманду /find <ключевое слово> <yyyy-mm-dd>. Например /find Tesla 2023-01-15
При получении запроса бот записывает chat_id и параметры запроса в хранилище Shelve
Если таких записей более 100, то в ответ отправляется сообщение о невозможности выполнить запрос.
Также пользователдь получает отказ для нового запроса, если в списке уже есть необработанный запрос от этого пользователя
Параллельно этому запускается бесконечный процесс обработчика, реализованный в отдельном процессе (thread).
Обработчик находит самый ранний зарегистированыый запрос и вызывает функцию пармера, передавая параметры поиска и chat ID,
который используется в качестве имени CSV файла результата поиска. Когда работа парсера заканчивается, файл результата
отправляется в соответствующий чат.
Затем запрос удаляется из списка и файл результата удаляется.

К сожалению, пока тестировался только на одном поьзователе Telegram.
