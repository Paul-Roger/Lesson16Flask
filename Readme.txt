NewsAutoRu parcer HTML interface
================================

Программа реализует интерфейс и парсер, котрые позволяют искать автомобильные новости по маркам автомобилей
Пока жтот вариант является однопользовательским, однозадачным. То есть, нельзя делать новый поиск, пока не завершен старый,
и результаты хранятся в виде файла CSV, который будет перезаписан при следующем поиске.
